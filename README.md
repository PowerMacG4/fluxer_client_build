# fluxer_client_build
Build Fluxer mobile app for iOS(unofficial)

## What is this?
On that day, They released open beta for Android, but they did not release for iOS.

I know why they do this, because the TestFlight has usable members of limit.

But I want to try official mobile app RIGHT NOW.

Then I thought build for iOS by myself

## Download
Pre-built IPA is available on [Actions Artifact](https://github.com/PowerMacG4/fluxer_client_build/actions).

## Install
Pre-built IPA is not signed, then You must sign IPA to install.

You can use [SideStore](https://github.com/SideStore/SideStore) and [LiveContainer](https://github.com/LiveContainer/LiveContainer) for IPA sign.
